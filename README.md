# Codex 文档导航

这是当前目录下的 Codex 中文资料入口，适合按需要快速跳转。

## 推荐阅读顺序

1. 先看 [codex_handbook.md](./codex_handbook.md)
2. 再按需要进入专项文档

## 文档说明

### [docs.html](./docs.html)

静态首页。适合在浏览器里直接打开，快速浏览这套 Codex 中文资料的入口关系。

### [codex_handbook.md](./codex_handbook.md)

总手册。适合第一次系统了解 Codex 时阅读，内容包含：

- 快速上手
- 核心概念
- 中文提示词模板
- 常见问题与排查
- 官方文档索引

### [codex.md](./codex.md)

使用指南。适合需要快速查安装、登录、命令、配置、权限模式时使用。

### [codex_concepts.md](./codex_concepts.md)

概念解释。适合系统理解这些关键词时使用：

- `agent`
- `model`
- `tool`
- `MCP`
- `skill`
- `AGENTS.md`
- `subagent`
- `sandbox`
- `approval`
- `automation`

### [prompt_template.md](./prompt_template.md)

提示词模板。适合直接复制使用，包含：

- 通用骨架
- 修 Bug 模板
- 做功能模板
- 代码审查模板
- 先澄清需求模板
- 解释代码模板

### [14_models.md](./14_models.md)

模型对比文档。适合查看状态页里的 14 条模型节点记录，包括：

- 实时状态总表
- 逐条优缺点
- 简短选型结论

## 你可以怎么用

### 想马上开始用 Codex

先看 [codex.md](./codex.md)。

### 想系统理解 Codex 的整体结构

先看 [codex_handbook.md](./codex_handbook.md)，再看 [codex_concepts.md](./codex_concepts.md)。

### 想直接复制提示词

看 [prompt_template.md](./prompt_template.md)。

### 想知道 `agent`、`skill`、`MCP` 到底有什么区别

看 [codex_concepts.md](./codex_concepts.md)。

### 想看 14 个模型节点的优缺点对比

看 [14_models.md](./14_models.md)。
